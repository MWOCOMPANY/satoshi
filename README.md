# **satoshi**
_____
## How to install (Termux) :iphone:
```
pkg update -y && pkg upgrade -y
pkg install python -y
git clone https://github.com/pixxxionix/satoshi.git
cd satoshi
python satoshi.py
```

## How to install (Kali) :computer:
```
sudo apt update
sudo apt upgrade -y
sudo apt install git python3 python3-pip
git clone https://github.com/pixxxionix/satoshi.git
cd satoshi
python3 satoshi.py
```
